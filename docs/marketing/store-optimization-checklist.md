## Description Template (Long)
```
Find and fix slow React components — directly in DevTools.

React Perf Profiler hooks into the React DevTools API to show you exactly which components are wasting renders, how long each commit takes, and what to optimize next.

KEY FEATURES:
• Flamegraph — identify bottlenecks at a glance
• Component Tree — visualize hierarchy and render counts
• Timeline — track commits over time
• AI Suggestions — get actionable optimization recommendations
• Performance Budgets — set thresholds, get alerts on violations
• Export — JSON, CSV, or HTML reports

ZERO CODE CHANGES REQUIRED. Install, open DevTools, and start profiling.

Works with React 16.8+ (hooks), React 18 (concurrent), and React 19. Supports Chrome (Manifest V3) and Firefox.

PRIVACY FIRST:
All profiling data stays in your browser. Optional cloud sync and AI features are opt-in only.
```
