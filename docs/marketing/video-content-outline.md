# Video Content Outline

## 1. Getting Started (2 min)
**Purpose:** Quick start for new users. Target: YouTube, landing page embed.
- 0:00-0:10 Hook: "Find wasted renders in your React app in 30 seconds"
- 0:10-0:30 Install from Chrome Web Store
- 0:30-0:50 Open DevTools, navigate to Perf Profiler tab
- 0:50-1:20 Start recording, interact with a demo app, stop recording
- 1:20-1:40 Walk through flamegraph — identify a slow component
- 1:40-2:00 Show optimization suggestion → apply fix → verify improvement

## 2. Deep Dive: Performance Analysis (5 min)
**Purpose:** Technical deep dive for experienced React developers.
- 0:00-0:30 Setup: demo app with intentional performance issues
- 0:30-1:30 Flamegraph: reading the chart, identifying bottlenecks
- 1:30-2:30 Component tree: render counts, wasted render detection
- 2:30-3:30 Timeline: correlating renders with user interactions
- 3:30-4:15 AI suggestions: getting optimization recommendations
- 4:15-5:00 Before/after comparison with Profile Compare feature

## 3. Advanced Features (5 min)
**Purpose:** Showcase pro features for power users.
- 0:00-1:00 Performance budgets: setting thresholds, getting alerts
- 1:00-2:00 Export workflows: JSON, CSV, HTML reports
- 2:00-3:00 Cloud sync: sharing profiles with the team
- 3:00-4:00 3D visualization: exploring the component tree in 3D
- 4:00-5:00 CI/CD integration: using the CLI for build-time checks

## 4. Social Clips (6 x 30-60 seconds)
**Purpose:** Short-form content for Twitter/X, LinkedIn, TikTok.
1. "This React component re-renders 47 times on page load" — show flamegraph discovery
2. "Add one line to fix 30 wasted renders" — show React.memo suggestion
3. "Profile your React app without changing a single line of code" — install + record
4. "This 3D visualization shows your entire React component tree" — show 3D viz
5. "Set a performance budget and get alerts when you exceed it" — show budget notification
6. "Export your profiling data as a beautiful HTML report" — show export

## Production Notes
- **Software:** OBS Studio (free) for screen recording
- **Resolution:** 1920x1080 minimum
- **Demo app:** Use a purposely slow React app (create one with known issues)
- **Audio:** Clear narration, professional mic
- **Thumbnails:** Use the flame gradient brand colors, bold text overlay
