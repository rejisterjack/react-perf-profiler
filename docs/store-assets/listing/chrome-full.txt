React Perf Profiler is a DevTools extension for analyzing React component performance: wasted renders, memoization, flamegraphs, timelines, and optional RSC-oriented views.

DEFAULT PRIVACY: Profiling runs locally in your browser. Data is stored in extension storage unless you export it yourself.

OPTIONAL FEATURES (user-configured only): Cloud sync (S3, Dropbox, Google Drive, etc.), team/collaboration, and cloud AI providers (OpenAI, Anthropic) can send data to third parties you choose. Local Ollama keeps AI on your machine. See the privacy policy linked from the store listing.

KEY CAPABILITIES:
- Wasted render detection and suggestions (memo, useMemo, useCallback)
- Interactive flamegraph and component tree
- Memo effectiveness analysis
- RSC-oriented analysis where applicable
- Web Worker-backed analysis for large sessions
- Export/import of profiles and optional CI budget tooling (perf-check CLI in repo)

FOR: React developers optimizing render cost, debugging re-renders, and learning render behavior.

Open source: https://github.com/rejisterjack/react-perf-profiler
