React Perf Profiler brings React performance analysis to Firefox DevTools: wasted renders, flamegraphs, memo analysis, and optional RSC-oriented tooling.

DEFAULT PRIVACY: Profiling stays on your device in extension storage until you export or enable optional features.

OPTIONAL: Cloud sync, collaboration, and cloud LLM features only send data when you configure them. See the privacy policy URL on the listing.

FEATURES:
- Wasted render detection with actionable hints
- Flamegraph and tree navigation
- Memo / callback stability insights
- Export profiles, keyboard shortcuts, dark mode
- Performance budget tooling via open-source CLI in the repository

Open source: https://github.com/rejisterjack/react-perf-profiler
