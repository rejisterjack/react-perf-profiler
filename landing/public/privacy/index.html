Privacy Policy — React Perf Profiler

Last updated: April 3, 2026

This page is the canonical privacy policy for store listings (Chrome Web Store and Firefox Add-ons). Markdown sources live in the same folder of the GitHub repository.

Who we are

React Perf Profiler is open-source software. Repository: github.com/rejisterjack/react-perf-profiler.

Default behavior (no optional features)

When you only record and analyze profiles in DevTools:

  • Profiling data (component names, render timings, commit structure from the React Profiler path) is processed in extension contexts on your machine.
  • Storage: Preferences and session data are kept in the browser’s extension storage (e.g. chrome.storage.local on Chrome; equivalent APIs on Firefox).
  • The project does not operate its own analytics server, advertising SDK, or account system for the extension.
  • The tool is aimed at React render metadata you generate while debugging, not at harvesting unrelated page content.

Optional features that can send data off your device

If you choose to enable and configure these features, data may leave your browser as described:

Feature What may be sent Where
Cloud sync (S3, Dropbox, Google Drive, etc.) Exported profile files or payloads you upload The cloud provider you configure (their terms apply).
AI suggestions (OpenAI, Anthropic, …) Prompt text derived from your analysis (you may use privacy options in the UI where available) and API authentication headers The provider’s API endpoints.
Ollama (local) Requests to your local Ollama instance Your machine only.
Team / collaboration (if enabled) Session or signaling data required for the feature Depends on configuration; see in-app description when enabled.

You control activation: Optional network features require your configuration.

Permissions (summary)

The extension may request broad host access so a content script and bridge can attach on development URLs you open (including iframes). That supports React / DevTools hook access—not unrelated data collection. See the repository’s permission justification doc for copy-paste text for reviewers.

Data retention and deletion

  • Clear extension data in the browser to remove stored profiles and settings.
  • Exported files on disk are your responsibility.
  • Cloud copies are managed in the provider you used.

Children’s privacy

The extension is intended for developers and is not directed at children under 13.

Changes

We may update this policy. The “Last updated” date will change when we do.
